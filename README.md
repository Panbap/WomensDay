# 🌸 Happy Women's Day

> 💖 Một trang web nhỏ gửi lời chúc tốt đẹp đến một nửa thế giới nhân ngày 8/3.

🌐 **Live Demo:** https://panbap.github.io/WomensDay/

---

## ✨ Giới thiệu

Đây là một project front-end đơn giản nhằm tạo ra một trang web chúc mừng **Ngày Quốc tế Phụ nữ (8/3)** với hiệu ứng đẹp mắt và ý nghĩa.

Phù hợp để:
- 💌 Gửi lời chúc đến người yêu / bạn bè / gia đình  
- 🎁 Làm landing page sự kiện  
- 🌸 Thực hành HTML, CSS, JavaScript
